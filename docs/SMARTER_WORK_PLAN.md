# SMARTER_WORK_PLAN — colors

תאריך: 2026-05-02

## ביקורת אמת

הפרויקט כבר חי ועובד ברמת בסיס, אבל כדי שהוא יהיה באמת כלי עבודה חכם ולא רק אתר דוגמאות, צריך להפוך אותו ל־Design Prompt Builder מסודר.

## מה לא מספיק חכם בגישה הקודמת

- חלק מהרכיבים נוספו בקבצי JavaScript נפרדים במקום מקור נתונים מרכזי.
- שכבת הגופנים והגדלים התחילה כנספח ולא כחלק מרכזי מהמערכת.
- אין עדיין דרך מספיק חכמה לבחור לפי שימוש: רקע, כותרת, כפתור, טבלה, גופן, גודל, אתר חינוכי, אתר פרימיום וכו׳.
- אין מספיק הפרדה בין נתונים לבין תצוגה.

## הכיוון החכם יותר

להוסיף שכבה חדשה בשם Smart Builder:

- `design-tokens.json` — מקור נתונים מרכזי לרכיבי עיצוב.
- `smart-builder.html` — דף בחירה נוח וממוקד.
- `smart-builder.css` — עיצוב נייד/נייח.
- `smart-builder.js` — חיפוש, סינון, תצוגה והעתקה.

## עקרונות חובה

- הכל בעברית RTL.
- כל כפתור אמיתי.
- כל Prompt שימושי ומדויק.
- כל צבע עם HEX/HSL.
- כל גופן עם שם CSS ו־fallback.
- כל גודל עם rem/px ושימוש מומלץ.
- אפשר להעתיק Prompt, CSS, שם/ערך.
- אפשר לבחור לפי שימוש ולא רק לפי קטגוריה.
- אין דמו ואין placeholder שמוצג כאמיתי.
