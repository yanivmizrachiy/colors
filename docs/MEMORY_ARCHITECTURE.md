- שער כניסה קצר מאוד.
- מקור אמת אחד לכל סוג מידע.
- מצב נוכחי קצר ונפרד.
- מסמכים כבדים לפי צורך בלבד.
- Rules ממוקדים לפי תחום או נתיב.
- Skills לתהליכים חוזרים.
- Hooks לכללים דטרמיניסטיים.
- Agents לחקירה, ביקורת ובדיקות נפרדות.
- בדיקות אוטומטיות במקום הוראות טקסט כשאפשר.
- מחקר והיסטוריה מחוץ להקשר הקבוע.
