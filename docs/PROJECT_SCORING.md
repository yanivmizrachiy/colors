# מודל דירוג ריפוים למחקר

## מטרה
לקבוע סדר מחקר לפי ערך אמיתי ולא לפי גודל בלבד.

## שיטת הציון
כל ריפו מקבל ציון 0–5 בכל ממד. הציון המשוקלל קובע את סדר המחקר, אך אינו מחליף שיקול מקצועי.

| ממד | משקל | שאלה |
|---|---:|---|
| פעילות עדכנית | 20% | כמה לאחרונה ובאיזו תדירות הריפו משתנה? |
| חשיבות ליניב | 20% | כמה הפרויקט מרכזי לעבודה, להכנסה או למטרות המקצועיות? |
| מורכבות טכנית | 15% | כמה שכבות, אינטגרציות, מסד נתונים, פריסה ואבטחה יש בו? |
| מעורבות Claude Code | 15% | כמה commits/PRs ותהליכי עבודה בוצעו באמצעות Claude Code? |
| כמות חיכוכים | 15% | כמה תיקוני המשך, רגרסיות, תקלות פרודקשן או אי-הבנות קיימים? |
| ערך לימודי חוצה-פרויקטים | 10% | כמה הממצאים צפויים לשפר גם פרויקטים אחרים? |
| איכות הראיות | 5% | האם קיימים PRs, בדיקות, תיעוד ונתוני שימוש שניתן לנתח? |

## נוסחה
`ציון כולל = Σ(ציון ממד × משקל)`

## קטגוריות
- **4.20–5.00 — עדיפות קריטית:** מחקר עומק מיידי.
- **3.40–4.19 — עדיפות גבוהה:** מחקר לאחר הקבוצה הקריטית.
- **2.50–3.39 — עדיפות בינונית:** מחקר ממוקד בנושאים רלוונטיים.
- **מתחת ל־2.50 — עדיפות נמוכה:** מחקר רק אם מופיע דפוס משותף או צורך חדש.

## כללי שימוש
1. יש לשמור ליד כל ציון את הראיה שהובילה אליו.
2. הציון מתעדכן כאשר מתקבל מידע חדש.
3. גודל ריפו לבדו אינו מעלה עדיפות.
4. ריפו קטן אך פעיל ובעייתי יכול להקדים ריפו גדול ויציב.
5. אין לבצע שינוי בריפו עקב הציון; הציון קובע מחקר בלבד.
