## 2. מטרת המוצר
פלטפורמה מסחרית עברית־תחילה ללימוד מתמטיקה עבור תלמידים ומורים. כוללת מאגר חומרים, משימות אינטראקטיביות, ניהול משתמשים והרשאות, מסד PostgreSQL, תשתיות קבצים, מייל ותשלומים, וכן פינה למורים פרטיים.

## 3. סטאק וארכיטקטורה
- Next.js 16.2 App Router, React 19, TypeScript strict.
- PostgreSQL + Drizzle ORM ומיגרציות.
- אימות והרשאות מותאמים אישית, sessions במסד ו-RBAC deny-by-default.
- Vercel לפריסה ול-preview של PRs.
- ספקי storage/email/payments מאחורי interfaces.
- בדיקות Node test runner שמנויות במפורש ב-`package.json`.
- pipeline פריסה כולל migrations, seeds ו-build דרך `vercel-build`.

## 4. מודל העבודה עם Claude Code

### רמת אוטונומיה
