# STYLE_CART — colors V7

## מה נוסף

V7 מוסיפה “חבילת הסגנון שלי”.

במקום להעתיק רכיב אחד בלבד, יניב יכול:
1. לבחור כמה רכיבים.
2. לראות אותם באזור חבילת הסגנון.
3. ללחוץ `העתק Prompt חבילה`.
4. להדביק ל־GPT הוראה אחת שמכילה את כל הבחירות.

## למה זה חשוב

זה הופך את colors מכלי בחירה נקודתי לכלי שמרכיב שפה עיצובית שלמה לאתר חדש.

## כלל אמת

החבילה משתמשת רק ברכיבים אמיתיים שקיימים ב־site.config.json.
