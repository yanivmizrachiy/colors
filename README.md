# קלוט-קוד

מרכז השליטה של יניב למחקר, תכנון ושיפור העבודה עם Claude Code.

## מטרת-העל
לבנות מערכת עבודה אישית, מקצועית ומתקדמת שמייעלת את כל מחזור העבודה עם Claude Code: הבנת פרויקט, תכנון, ביצוע, בדיקות, Git, PR, CI, פריסה, תיעוד, אבטחה, אוטומציה, המשכיות בין שיחות ומחשבים, בחירת מודלים וכלים — ובמקביל מצמצמת זמן, טעויות וטוקנים.

## גבול מחייב
ריפוים אחרים משמשים למחקר בקריאה בלבד. אין לשנות אותם בלי הוראה מפורשת ונפרדת של יניב.

## מבנה
- `CLAUDE.md` — שער כניסה קצר ומחייב.
- `PROJECT_GOALS.md` — מטרות ומדדים.
- `CURRENT_STATE.md` — מצב נוכחי קצר.
- `RESEARCH_LOG.md` — יומן מחקר.
- `DECISIONS.md` — החלטות שאושרו.
- `docs/` — ארכיטקטורה, אבטחה, מחקר ותוכנית עבודה.
