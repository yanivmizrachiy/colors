# תבנית משימה קצרה ל־Claude Code

## תבנית רגילה
```text
משימה: [תוצאה אחת ברורה].
היקף: [קבצים/אזור שמותר לשנות].
לא לשנות: [אזורים מוגנים].
הצלחה: [תוצאה מדידה או נראית].
בדיקות: [static / build / preview / environment / live].
Git: עבוד בענף ופתח PR; אל תמזג לפני דיווח.
```

## תוספת למשימה בינונית
```text
לפני שינוי הצג תוכנית קצרה: קבצים צפויים, סיכונים, קריטריוני קבלה ובדיקות. לאחר מכן המשך בלי לעצור אלא אם נדרש שינוי מחוץ להיקף.
```

## תוספת למשימה גדולה או מסוכנת
```text
תחילה חקור בלבד. הפרד עובדות מהשערות, הצג root cause/חלופות/rollback, ופצל את העבודה לשלבים עצמאיים. אין שינוי ב־DB, auth, deployment, secrets או מידע אמיתי בלי אישור נפרד.
```

## בקשת סיום קבועה
```text
בסיום דווח:
1. מה בוצע.
2. מה נבדק מקומית.
3. מה אומת בסביבה חיה.
4. מה לא אומת או חסום.
5. branch, commit ו־PR.
6. rollback והצעד הבא.
```
